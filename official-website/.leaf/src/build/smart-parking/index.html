          基于5G NB-IoT技术的

          智慧停车解决方案

          领翌车位检测器采用自主研发的毫米波雷达+地磁双模传感器，掌握核心算法，同步采用NB-IoT技术对车位进行管理，构建客户端及管理平台，将海量的停车位进行数字化智慧管理。目前智慧停车系统已在珠海、烟台、南京、常州、无锡、连云港等地实现业务布局

          核心竞争力
