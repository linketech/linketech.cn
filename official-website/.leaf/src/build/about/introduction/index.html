          公司介绍

          领翌技术（横琴）有限公司成立于2018年6月，注册资本3000万元。总部位于珠海横琴自贸区，在成都、南京、北京及加拿大滑铁卢设有研发中心。

          领翌技术是拥有核心技术及完全自主知识产权的高科技创新企业，成立至今拥有授权专利50项。创始人漆一宏院士（珠海市顶尖人才）及其核心团队通过多年理论研究与实践，首次发现了颠覆教科书的紧耦合无线收发机理论与技术（全球400余项授权专利，100余篇权威杂志论文）。基于上述核心无线理论与技术，企业搭建“高灵敏度接收机”、“低噪声发射机”、“高性能天线”及“高速连接器”组成的产品赋能平台，实现大容量、高速率、广覆盖、稳定、可靠的新一代高性能无线产品及系统解决方案，推进整个赋能平台的普适性、泛在性、智能性、环保性发展。

          2018年，领翌技术在横琴首届创新创业大赛中脱颖而出；2019年，成为GTI（全球性通信协会）合作成员、中国移动5G联合创新中心合作伙伴；入选珠海市创新创业团队及广东省“众创杯"科技海归领航赛金奖。目前团队正在对“APS毫米级差分定位”、“高速Type-C数据线”、“低延时游戏加速路由器”、“高灵敏卫星定位终端及系统”及“5G NB-IoT车辆智能检测系统”等多类无线产品进行赋能。基于自主无线电技术、核心收敛算法，打造低成本高性能赋能产品，助推无线产业转型升级。
